Klassen KeyAdapter
import se.lth.cs.pt.dots.listeners.KeyAdapter;

En 'tom' klass som implementerar interfacet KeyListener. Genom att ärva från denna klass slipper vi deklarera alla KeyListener-operationer om vi bara är intresserad av någon enstaka av dem.

Parametrar:
e:information om tangenttryckningen.
