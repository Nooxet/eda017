Klassen StopWatch
import se.lth.cs.pt.clock.StopWatch;

Används för att mäta tid i program, kan mäta tider med en högsta upplösning på 1 millisekund (noggrannheteten är dock inte nödvändigtvis riktigt så god).

Exempel på användning: 

     StopWatch sw = new StopWatch();
     sw.reset();
     sw.start();
     // programrader vi skall ta tiden på
     sw.stop();
     System.out.println("Tid: " + sw.getValue() + " ms.");

Konstruktorer
StopWatch  ()
Skapar en ny klocka som är stoppad och nollställd.
Metoder
void start()
Startar klockan, om klockan redan går händer ingenting.
void restart()
Startar om klockan.
void stop()
Stoppar klockan, om klockan redan är stoppad händer ingenting.
void reset()
Nollställer klockan, om klockan inte är stoppad fortsätter den räkna (men börjar i så fall om från 0).
int getValue()
Returnerar antal millisekunder som klockan räknat sedan den senast nollställdes.
