Klassen MouseAdapter
import se.lth.cs.pt.graphics.listeners.MouseAdapter;

En 'tom' klass som implementerar gränssnittet MouseListener. Genom att ärva från denna klass slipper vi deklarera alla MouseListener-operationer om vi bara är intresserad av någon enstaka av dem.

Konstruktorer
MouseAdapter  ()
Metoder
void mouseClicked(MouseEvent e)
Anropas om användaren klickar med musen.
Parametrar:
e:den aktuella mushändelsen.
void mousePressed(MouseEvent e)
Anropas om användaren trycker ner mus-knappen.
Parametrar:
e:den aktuella mushändelsen.
void mouseReleased(MouseEvent e)
Anropas om användaren släpper upp mus-knappen.
Parametrar:
e:den aktuella mushändelsen.
void mouseMoved(MouseEvent e)
Anropas om användaren flyttar musen.
Parametrar:
e:den aktuella mushändelsen.
void mouseDragged(MouseEvent e)
Anropas om användaren flyttar musen medan en musknapp är nedtryckt.
Parametrar:
e:den aktuella mushändelsen.
