Gränssnittet MouseListener
import se.lth.cs.pt.graphics.listeners.MouseListener;

Beskriver ett objekt som hanterar olika slags mus-händelser.

Metoder
void mouseClicked(MouseEvent e)
Anropas om användaren klickar med musen.
Parametrar:
e:ett MouseEvent som beskriver händelsen.
void mousePressed(MouseEvent e)
Anropas om användaren trycker ner en musknapp.
Parametrar:
e:ett MouseEvent som beskriver händelsen.
void mouseReleased(MouseEvent e)
Anropas om användaren släpper upp en musknapp.
Parametrar:
e:ett MouseEvent som beskriver händelsen.
void mouseMoved(MouseEvent e)
Anropas om användaren flyttar musen.
Parametrar:
e:ett MouseEvent som beskriver händelsen.
void mouseDragged(MouseEvent e)
Anropas om användaren drar musen medan någon knapp är nedtryckt.
Parametrar:
e:ett MouseEvent som beskriver händelsen.
