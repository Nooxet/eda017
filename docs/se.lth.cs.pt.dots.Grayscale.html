Klassen Grayscale
import se.lth.cs.pt.dots.Grayscale;

Beskriver olika gråskalor. Detta är egentligen ett slags Color-objekt, men färgen är alltid en ren gråskala (dvs det finns lika mycket rött, grönt och blått i den).

Konstruktorer
Grayscale  (int level)
Skapar ett nytt Grayscale-objekt.
Parametrar:
level:graden av svärta - lägg märke till att 0 = svart och 255 = vit!
Metoder
int getLevel()
Ger graden av svärta (0 = svart, 255 = vit).
