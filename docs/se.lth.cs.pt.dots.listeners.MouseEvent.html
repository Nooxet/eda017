Klassen MouseEvent
import se.lth.cs.pt.dots.listeners.MouseEvent;

Beskriver en mushändelse av något slag.

Metoder
DotWindow getSource()
Talar om vilket fönster som händelsen inträffade i.
Returnerar det aktuella fönstret.
int getX()
Ger x-koordinaten för händelsen (alltid ett heltal).
Returnerar x-koordinaten.
int getY()
Ger y-koordinaten för händelsen (alltid ett heltal).
Returnerar y-koordinaten.
int getButton()
Ger knapp-numret. På en Linux-dator kan värdet vara:
  • 0: ingen knapp
  • 1: vänster musknapp
  • 2: höger knapp
  • 3: tredje knappen (om den finns)
Returnerar knappens nummer.
long getTime()
Ger tidpunkten för händelsen.
Returnerar Returnerar tid mätt i millisekunder (sedan 1 januari 1970).
