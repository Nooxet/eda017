Klassen ImageReader
import se.lth.cs.pt.util.ImageReader;

Denna klass används bara för att hämta bilder från filer. Du kan hämta en bild genom att skriva: 

   Image image = ImageReader.getImage("picture.png");

Klassmetoder
Image getImage(String filename)
Hämtar en bild från en fil (som måste vara av formatet .png, .jpg (.jpeg) eller .gif - ger exekveringsfel om filen som vi försöker hämta bilden från inte finns. Den inlästa bilden lagras i ett Image-objekt.
Parametrar:
filename:namnet på filen med bilden.
Returnerar ett Image-objekt
