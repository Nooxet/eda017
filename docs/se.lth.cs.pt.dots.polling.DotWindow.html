Klassen DotWindow
import se.lth.cs.pt.dots.polling.DotWindow;

Denna klass kan, förutom det som alla DotWindow-klasser kan (rita olika figurer), även låta oss veta vad användaren gör. Vi får veta detta genom att anropa getState()-operationen, som returnerar ett State-objekt som beskriver det som hänt sedan senaste getState()-anropet.

   DotWindow w = new DotWindow(...);
   Clock clock = new Clock();
   int x = ...;
   int y = ...;
   while (...) {
       clock.pause(...);
       State state = w.getState();
       if (state.isMouseClicked()) {
           x = state.getMouseX();
           y = state.getMouseY();
       }
       if (state.isKeyPressed('q')) {
           break;
       }
   }

Konstruktorer
DotWindow  (int xSize, int ySize, int dotSize)
Skapar ett nytt DotWindow med given bredd, höjd och prickstorlek.
Parametrar:
xSize:antal prickar i x-led.
ySize:antal prickar i y-led.
dotSize:prickarnas bredd och höjd, i pixlar.
Metoder
State getState()
Ger de nya händelser som inträffat sedan vi anropade operationen förra gången.
Returnerar information om nedtryckta tangenter eller mushändelser (se klassen State).
