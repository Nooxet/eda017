Klassen DotWindow
import se.lth.cs.pt.dots.events.DotWindow;

Detta är ett slags DotWindow som utöver de vanliga DotWindow-operationerna även låter oss registrera vilka händelser vi vill att vårt program skall ta hand om. Exempel på användning (se även klassen GameEvent):

   DotWindow w = ...;
   ...
   w.checkKeys(true,false,false);
   w.checkMouse(true,false,false,false,false);
   w.timeStep(100);
   for (;;) {
       GameEvent event = w.getNextEvent();
       switch (event.getKind()) {
       case GameEvent.KEY_PRESSED:
           // Användaren har tryckt ner en tangent
           keyPressed(event.getKey());
           break;
       case GameEvent.MOUSE_CLICKED:
           // Användaren har klickat med musen
           mouseClicked(event.getX(), event.getY());
           break;
       case GameEvent.TICK:
           // Det har gått ytterligare 100 millisekunder
           tick();
           break;
       }
   }

Ritoperationerna dokumenteras här.

Konstruktorer
DotWindow  (int xSize, int ySize, int dotSize)
Skapar ett nytt DotWindow med given bredd, höjd och prickstorlek.
Parametrar:
xSize:antal prickar i x-led.
ySize:antal prickar i y-led.
dotSize:prickarnas bredd och höjd, i pixlar.
Metoder
void checkKeys(boolean press, boolean release, boolean type)
Talar om vilken slags tangentbordshändelser vi är intresserade av.
Parametrar:
press:om true skall vi lyssna på tangenttryck
release:om true skall vi lyssna på tangentsläpp
type:om true skall vi lyssna på tangenttryck (dvs en fullbordad tangenttryckning).
void checkMouse(boolean click, boolean press, boolean release, boolean drag, boolean move)
Talar om vilka mushändelser vi är intresserade av.
Parametrar:
click:om true skall vi bry oss om musklick, dvs att användaren trycker ner och släpper upp musknappen ganska snabbt på ett och samma ställe
press:om true skall vi bry oss om ifall användaren trycker ned musknappen
release:om true skall vi bry oss om ifall användaren släpper upp musknappen
drag:om true skall vi bry oss om ifall användaren flyttar musen med musknappen nedtryckt.
move:om true skall vi bry oss om ifall användaren flyttar musen utan att trycka på någon knapp.
void timeStep(int ms)
Gör att vi får nya GameEvent-objekt med jämna tidsintervaller när vi anropar getNextEvent().
Parametrar:
ms:antalet millisekunder mellan varje GameEvent, om antalet millisekunder är 0, eller negativt, så kommer fönstret att sluta generera GameEvent-objekt tills vi åter anropar operationen med ett positivt värde
void startClock(int ms)
Startar klockan.
Parametrar:
ms:antalet millisekunder mellan varje tick.
void stopClock()
Stoppar klockan.
GameEvent getNextEvent()
Ger nästa händelse som vi sagt att vi är intresserade av (beroende på om och hur vi anropat operationerna checkKeys, checkMouse och timeStep.
