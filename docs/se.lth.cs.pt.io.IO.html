Klassen IO
import se.lth.cs.pt.io.IO;

Man kan inte skapa något IO-objekt. Istället använder man klassen för att skapa Scanner- eller PrintStream-objekt på följande sätt: 

 Scanner keyboard = IO.keyboardScanner();
 PrintStream console = IO.consoleWriter();
 int n = keyboard.nextInt();
 console.println("Värdet på n är " + n);

 // Kopiera texten i input.txt till output.txt:
 Scanner in = IO.fileScanner("input.txt");
 PrintStream out = IO.fileWriter("output.txt");
 while (in.hasNextLine()) {
     out.println(in.nextLine());
 }
 out.flush();
 out.close();

Klassmetoder
Scanner keyboardScanner()
Ger ett Scanner-objekt som läser från tangentbordet - du kan få samma effekt genom att skriva: 
       Scanner s = new Scanner(System.in);
Returnerar en referens till ett Scanner-objekt.
Scanner fileScanner(String filename)
Ger ett Scanner-objekt som läser från en given textfil, returnerar null om ingen sådan fil finns.
Parametrar:
filename:namnet på filen vi skall läsa från.
Returnerar en referens till ett Scanner-objekt, eller null (om filen inte finns).
PrintStream consoleWriter()
Ger en referens till ett PrintStream-objekt som skriver ut i kommandofönstret.
Returnerar en referens till ett PrintStream-objekt.
PrintStream fileWriter(String filename)
Ger en referens till ett PrintStream-objekt som skriver ut till en fil med ett givet namn.
Parametrar:
filename:namn på filen vi skall skriva till.
Returnerar en referens till ett PrintStream-objekt.
