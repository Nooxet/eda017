Klassen Debug
import se.lth.cs.pt.debug.Debug;

Denna klass låter oss enkelt göra testutskrifter som kan slås till och från med Debug.on() och Debug.off()-operationerna.

Ett enkelt exempel: 
     Debug.println("Detta är en debug-utskrift");
     Debug.println("str = " + str);
     Debug.printf("n = %d, p = %d\n", n, p);
     ...
     // Vi kan stänga av testutskriferna genom att skriva:
     Debug.off();
     Debug.println("Denna utskrift kommer inte att göras...");
     ...
     // Om vi åter vill slå på testutskriferna kan vi skriva:
     Debug.on();
     Debug.println("Nu syns testutskrifterna igen...");
     ...

Testutskrifterna är automatiskt tillslagna när programmet startar.

Poängen med klassen är att vi kan göra en massa testutskrifter som vi tittar på medan vi utvecklar våra program, och sedan stänga av alla utskrifter genom att bara skriva Debug.off() i början av huvudprogrammet (vi behöver alltså inte ta bort våra testutskrifter).

Klassmetoder
void on()
Gör att testutskriferna syns tills vi eventuellt anropar Debug.off().
void off()
Stänger av testutskriferna tills vi eventuellt anropar Debug.on().
void setPrefix(String prefix)

Ger alla följande testutskrifter ett givet prefix, om vi vill att alla testutkrifter skall börja med texten Testutskrift: så anropar vi: 

Debug.setPrefix("Testutskrift: ");
. För att ta bort prefixet skriver vi bara: 
Debug.setPrefix("");

Parametrar:
prefix:den text vi fill ha först i alla testutskrifter
void usePausing(boolean willPause)

Gör det möjligt att bestämma om programmet skall vänta på att användaren trycker på "Return"-tangenten efter varje testutskrift.

Parametrar:
willPause:true om programmet skall vänta, false annars.
void printf(String pattern, Object[] arguments)

Gör en testutskrift, fungerar ungefär som System.out.printf-operationen, men ger bara utskrifer så länge testutskriferna är tillslagna.

Parametrarna ovan ser lite konstiga ut (deklarationen utnyttjar en finess i Java 5), vi skickar in alla värden som skall skrivas ut med komma-tecken emellan, inte som en Object-vektor. För att göra en testutskrift med värdet på e och pi med 5 decimaler kan vi skriva:

 Debug.printf("e = %.5d, pi = %.5d\n",
 Math.exp(1), Math.PI);
Parametrar:
pattern:formatet på det som skall skrivas ut
arguments:de värden som skall skrivas ut
void print(String msg)
Gör en testutskrift, bryter inte till ny rad.
Parametrar:
msg:den text som skall skrivas ut
void println(String msg)
Gör en testutskrift, bryter till ny rad efter utskriften.
Parametrar:
msg:den text som skall skrivas ut
void println()
Bryter till ny rad på testutskrifterna.
void printTo(PrintStream out)
Gör att testutskrifterna hamnar på en given enhet. Standardinställningen är att testutskriferna hamnar på System.err.
Parametrar:
out:den utskriftsenhet vi vill styra utskrifterna till
void printTo(String filename)
Gör att testutskrifterna hamnar på en fil med ett givet namn.
Parametrar:
filename:namnet på filen som skall innehålla testutskrifterna
