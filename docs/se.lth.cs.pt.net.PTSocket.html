Klassen PTSocket
import se.lth.cs.pt.net.PTSocket;

Med hjälp av klassen PTSocket kan vi låta två datorer kommunicera med varandra på ett enkelt sätt. 

   String url = "hacke-1.student.lth.se";
   int port = 1729;
   PTSocket socket = new PTSocket(url, port);
   if (socket.connect()) {
       socket.out.println("Det första meddelandet...");
   }
   while (socket.isOpen()) {
       String msg = socket.in.nextLine();
       socket.out.println("... mitt nästa meddelande ...");
   }
En av de båda datorer som sätt i förbindelse med varandra måste ta initiativet, och skicka det första meddelandet -- lämpligt är att låta den av de båda datorerna som får true tillbaka från connect-anropet skicka det första meddelandet.

Konstruktorer
PTSocket  (String url, int port)
Skapar förbindelse på given port till en dator med given url.
Parametrar:
url:den andra datorns adress
port:aktuellt portnummer
Metoder
boolean connect()
Startar förbindelsen, returnerar true om detta program skall skicka första meddelandet, false annars.
boolean isOpen()
Testar om förbindelsen fortfarande är öppen. Denna operation är ännu inte implementerad -- just nu returnerar den alltid true.
boolean close()
Stänger förbindelsen. Denna operation är ännu inte implementerad -- just nu returnerar den alltid true.
