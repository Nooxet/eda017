Klassen Die
import se.lth.cs.pt.die.Die;

Detta är den tärningsklass som beskrivs i föreläsningsanteckningarna. Den kan användas för att simulera tärningsspel.

Exempel på användning (räkna summan av 10 tärningskast): 

    int sum = 0;
    Die d = new Die();
    for (int i = 1; i <= 10; i++) {
        d.roll();
        sum += d.getNbrOfSpots();
    }
    System.out.println("Summan blev " + sum);

Konstruktorer
Die  ()
Skapar en sexsidig tärning. Tärningen måste kastas innan vi kan läsa av antal prickar.
Die  (int nbrOfSides)
Skapar en tärning med ett godtyckligt antal sidor. Tärningen måste kastas innan vi kan läsa av antal prickar.
Parametrar:
nbrOfSides:antalet sidor.
Metoder
void roll()
Kastar tärningen.
int getNbrOfSpots()
Ger antalet prickar som visas av tärningen efter senaste kastet.
int getNbrOfSides()
Ger antalet sidor på tärningen. För en tärning som är skapad med new Die() är detta alltid 6.
