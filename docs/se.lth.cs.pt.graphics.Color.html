Klassen Color
import se.lth.cs.pt.graphics.Color;

Beskriver de färger som vi kan använda i figurer som ritas i ett GraphicsWindow.

Färgerna beskrivs av vilka nyanser av rött, grönt och blått de har. Färgen vit innehåller maximalt av alla tre färgerna, färgen svart innehåller ingen av färgerna. Om vi blandar de tre färgerna kan vi få en mängd (16 miljoner) olika färgnyanser -- exempelvis består gult av (255,255,0), dvs det är en blandning av rött och grönt, utan något som helst inslag av blått.

Det finns ett antal färdigdefinierade färger:

  • Color.WHITE
  • Color.BLACK
  • Color.RED
  • Color.GREEN
  • Color.BLUE
Ett enkelt exempel på hur färgerna kan blandas och användas i ett GraphicsWindow: 
    Color blue = Color.BLUE;
    Color yellow = new Color(255,255,0);
    GraphicsWindow w = new GraphicsWindow(500,500);
    Rectangle rect = new Rectangle(150,150,40,40);
    rect.setBorderColor(blue);
    rect.setFillColor(yellow);
    w.add(rect);

Konstruktorer
Color  (int red, int green, int blue)
Skapar en ny färg med givet inslag av rött, grönt och blått.
Parametrar:
red:nyansen rött (0-255)
green:nyansen grönt (0-255)
blue:nyansen blått (0-255)
Color  (String htmlColor)
Skapar en ny färg med given html-kodning (dvs en sträng på formatet #RRGGBB", där RR, GG och BB är hexadecimala värden på nyanserna av rött, grönt och blått (dvs de är tal i intervallet 00..FF). Man kan utelämna #-tecknet i färgkodningen.
Parametrar:
htmlColor:html-kodningen
Metoder
int getRed()
Ger nyansen rött i denna färg (0-255).
int getGreen()
Ger nyansen grönt i denna färg (0-255).
int getBlue()
Ger nyansen blått i denna färg (0-255).
boolean equals(Object other)
Jämför om färgen är exakt lik en annan färg. Observera att vi inte bör jämföra färger med hjälp av ==-operationen: 
    Color c1 = new Color(10,20,30);
    Color c2 = new Color(10,20,30);
    if (c1 == c2) {
        // denna test misslyckas, eftersom c1 och c2 visserligen beskriver
        // samma färg, men är olika färg-objekt
    }
    if (c1.equals(c2)) {
        // denna test lyckas
    }
Parametrar:
other:den andra färgen
Returnerar true om de båda färgerna är lika, false annars.
