Klassen KeyEvent
import se.lth.cs.pt.dots.listeners.KeyEvent;

Beskriver en tangentbordshändelse.

Metoder
char getKey()
Talar om vilken tangent det handlar om (vi får tillbaka det tecken som står på tangenten). Denna operation fungerar inte för specialtangenter som exempelvis pil-tangenterna.
Returnerar tecknet som står på tangenten.
int getKeyCode()
Talar om vilken tangent det handlar om (vi får tillbaka koden för tangenten, som är ett heltal). Denna operation kan vara användbar om vi vill använda exempelvis pil-tangenterna.
Returnerar koden för tangenten.
long getTime()
Ger tidpunkten för händelsen.
Returnerar tid mätt i millisekunder (sedan 1 januari 1970).
